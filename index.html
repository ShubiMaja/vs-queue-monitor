VS Queue Monitor — Help
Picking the log
Use Edge or Chrome. The app reads the file you choose and polls for new lines as the log grows.
If the picker blocks protected folders
Browsers often refuse logs under AppData and similar. Put a reachable copy of the path under your profile: on Windows run the generated mklink /H in Command Prompt (same drive as Documents); on Linux/macOS use a ln -s under home. Paste your real path below — the generator fills exact commands and the pick target.
Windows: don’t pick Explorer “shortcuts” (.lnk) — they aren’t the log file. mklink /H applies to a file only; for folders the generator can offer mklink /J. Junction edge cases, empty folders, and PowerShell quirks are covered in the README.
