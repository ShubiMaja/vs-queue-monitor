Getting started

          
Set the Vintage Story data folder in the header (paste a path or use the folder icon), then click Start. The app watches client-main.log for queue position changes.

          
Typical log paths

          
    
            
  • Windows: %APPDATA%\VintagestoryData
    • 
            
  • macOS: ~/Library/Application Support/VintagestoryData
    • 
            
  • Linux: ~/.config/VintagestoryData
    • 
          

          
Threshold alerts

          
Alerts fire when position drops below a threshold — once per threshold per queue run. Add or edit thresholds in the Warnings KPI cell (+ or ✎).

          
Graph

          
Drag or pinch to zoom a time range. Scroll to zoom in/out. Click 1× to reset. Toggle trendline or live-follow with the top-right toolbar buttons. REL/ABS and LIN/LOG are in the footer next to the zoom controls.

          
Keyboard shortcuts

          
Space start/stop · o settings · F1 help · v copy history · + zoom in · − zoom out · 0 reset zoom. Ignored while a dialog is open.
