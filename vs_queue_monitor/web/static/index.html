            This UI runs in your browser but the Python app reads log files from disk. Paste the
            folder that contains client-main.log (or your data directory), or use
            The folder and log file icons in the header open a system dialog on this computer
            (requires Python with Tk). Then Start.

  • Linux: ~/.config/VintagestoryData
