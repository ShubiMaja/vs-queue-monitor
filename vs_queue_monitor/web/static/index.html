Log source path

        

          Paste the folder that contains client-main.log, your VintagestoryData folder, or a full path to a .log file. You can also use the folder or file buttons in the header.
        

        
        
        

          
          
        

      

    


    

      

      

        
Help

        

          

            This UI runs in your browser but the Python app reads log files from disk. Paste the
            folder that contains client-main.log (or your data directory), or use the
            folder and log file icons in the header to open a system dialog on this computer
            (requires Python with Tk). Then click Start.
          

          
Typical paths

          
    
            
  • Windows: %APPDATA%\VintagestoryData or ...\VintagestoryData\Logs
    • 
            
  • macOS: ~/Library/Application Support/VintagestoryData
    • 
            
  • Linux: ~/.config/VintagestoryData
    • 
          

          
Keyboard

          

            When focus is not in a text field: Space start/stop · o settings ·
            F1 this help · v copy session history
            (clipboard) · + zoom in graph · − zoom out · 0 reset zoom.
            Mouse wheel also zooms the graph. Single-key shortcuts are ignored while a dialog, the guided tour, or the restore banner is open.
          

          
Threshold alerts

          
Alerts fire when your queue position drops below a threshold value, once per threshold per run — not when it simply reaches that number. Add thresholds in the Warnings KPI (+ or ✎).
